# Boquila crypto library

## prerequisites

- [nodejs](https://nodejs.org/en) >= 18
- [circom](https://docs.circom.io/getting-started/installation/) >= 2.1.6
- [snarkjs](https://www.npmjs.com/package/snarkjs) >= 0.7

## Install dependencies

```sh
npm install
```

## Build the library

```sh
npm run build
```

## Run tests

```sh
npm test
```
